amo-cli

A CLI tool used to browse Mozilla Add-ons (AMO)

npm i -g amo-cli

amo-cli was created to scratch an itch I had for a long time - wanting to make my own CLI app. Being familiar with Mozilla Add-ons, I decided I would make a tool to interact with the AMO API, and that was how amo-cli came to be.
